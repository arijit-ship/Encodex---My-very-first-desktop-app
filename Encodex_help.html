Help and how it works!

 

 

This application is pretty straight forward. You have to give some input values(integer or string) using the virtual keypad available in the application and to get the corresponding ASCII, binary or hexadecimal values , use any of these buttons - - Get ASCII Value, Get Binary Value & Get Hex Value. 

 

If the given input contained only number(s), it will be considered as integer. Decimal point numbers will be treated as string.

For example-- if the input is  1258  , it will be considered as integer. But if you give input as 1258.00 , it will be treated as string of numeric and special characters and the output will be different for both these input.

 

 

Now let’s dig into some details!

 

 

l Input is integer: 

 

 

 

If the given input is an integer:

 

Get Binary Value  will return the corresponding binary value.

Get Hex Value    will return the corresponding hexadecimal value.

 

But for Get ASCII Value, there can be two possible cases-- the given integer may be a one-digit integer or multi-digit integer.

  

Now for one-digit integer, Get ASCII Value will return the corresponding ASCII value for that number key.

But for multi-digit integer, each digit will be considered independently and Get ASCII Value will return the concatenated ASCII value of each digit.

 

For example-

 

Input is 12. So here Get ASCII Value will return concatenated ASCII form of both 1 and 2. Output will be 4950 (49 for 1 and 50 for 2).

 

 

 

l Input is string: 

 

 

If the given input is a string of alphanumeric characters, each number will be considered as character as well along the other character (alphabet or special char).

 

For a single character, Get ASCII Value will return the corresponding ASCII value of the character. But again, for a string of multiple characters(including digits), it will return concatenated ASCII value of each character (including ASCII of the numeric keys).

 

Now come to the other two operation buttons for this situation.

 

For a string of single character:

 

Get Binary Value will return the binary value of corresponding ASCII integer of the given character. For example, if the given input is ‘A’, Get Binary Value will return the binary value of the ASCII integer of ‘A’. So the output will be 1000001, which is binary value of 65 (ASCII value of ‘A’ is 65)

 

This algorithm also goes true for Get Hex Value.

For for Get Hex Value output will be 41, which is hexadecimal value of 65(ASCII value of ‘A’).

 

 

 

For a string of multiple characters:

 

Get Binary Value will return the concatenated binary values of ASCII integers for each character(again including digit) in the string. For example, consider the input is ‘A12’. Here output will be 1000001110001110010. For ‘A’, ASCII value is 65 and binary value of 65 is 1000001. For ‘1’, ASCII value is 49 and binary value of 49 is 110001. And for ‘2’, ASCII value is 50 and binary value of 50 is 110010.

 

Same goes for Get Hex Value. For ‘A12’, output returned by Get Hex Value is 413132. Similarly, 41 is hexadecimal value of 65(ASCII of ‘A’), 31 is hexadecimal value of 49(ASCII of ‘1’) and 32 is hexadecimal value of 50(ASCII of 2).
